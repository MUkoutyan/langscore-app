# Langscoreについて    {#about}

LangscoreはRPGツクール VXAce, RPGツクール MV/MZで翻訳機能を実装するための補助ソフトです。

## 特徴

* 翻訳文をCSVファイルで記述します。
  - Excel, Googleスプレッドシートで編集することが出来ます。
* ゲーム中で即座に言語の切り替える事ができます。
  - 言語ごとにゲームファイルを分ける必要がありません。
* 言語に応じて、表示する画像を切り替えることが出来ます。
